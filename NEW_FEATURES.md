# תכונות חדשות במערכת Views

## 🆕 מה חדש?

### 1️⃣ **פילטרים לפי תאריכים** 📅

עכשיו אפשר להוסיף פילטר תאריכים לכל view!

**אופציות:**

- 🗓️ **כל הזמנים** - ללא פילטר
- 📅 **7 ימים אחרונים** - שבוע אחרון
- 📆 **30 ימים אחרונים** - חודש אחרון
- 🔧 **טווח מותאם אישית** - בחר מתאריך עד תאריך

**איך להשתמש:**

1. פתח "צור תצוגה חדשה"
2. גלול למטה ל"פילטר לפי תאריך"
3. לחץ ON
4. בחר שדה תאריך (createdAt, updatedAt, או שדה תאריך מהטבלה)
5. בחר טווח תאריכים
6. אם בחרת "טווח מותאם" - בחר תאריכים ספציפיים

### 2️⃣ **שדות מרובים** (בפיתוח)

התשתית מוכנה לתמיכה במספר שדות מספריים לחישוב!
יבוא בגרסה הבאה.

## 📊 דוגמאות שימוש

### דוגמה 1: לידים מהשבוע האחרון

```
סוג: סטטיסטיקות לפי זמן
טווח: השבוע הנוכחי
פילטר תאריך: ON
  - שדה: createdAt
  - טווח: 7 ימים אחרונים
```

### דוגמה 2: סכום הכנסות מחודש אחרון

```
סוג: חישובים ואגרגציות
חישוב: סכום
שדה: amount (סכום)
פילטר תאריך: ON
  - שדה: createdAt
  - טווח: 30 ימים אחרונים
```

### דוגמה 3: חלוקה לפי סטטוס - רק מינואר 2025

```
סוג: חישובים ואגרגציות
חישוב: קיבוץ
שדה לקיבוץ: status
פילטר תאריך: ON
  - שדה: createdAt
  - טווח: טווח מותאם
  - מתאריך: 2025-01-01
  - עד תאריך: 2025-01-31
```

## 🔧 שינויים טכניים

### קבצים שעודכנו:

1. `app/actions/views.ts` - ViewConfig עם dateFilter
2. `lib/viewProcessor.ts` - לוגיקה לפילטור תאריכים
3. `components/AddViewModal.tsx` - UI לבחירת תאריכים

### ViewConfig החדש:

```typescript
{
  dateFilter?: {
    field: string;           // שדה התאריך לפילטר
    type: "week" | "month" | "custom" | "all";
    startDate?: string;      // לטווח מותאם
    endDate?: string;        // לטווח מותאם
  }
}
```

## ✅ מה עובד

- ✅ פילטר תאריכים חכם
- ✅ זיהוי אוטומטי של שדות תאריכים
- ✅ תמיכה ב-createdAt/updatedAt
- ✅ טווחים מוגדרים מראש
- ✅ בחירת תאריכים מותאמת אישית
- ✅ שילוב עם פילטרים רגילים
